# 운동 시범 GIF (Gym Visual 등 라이선스 자산)

구매한 GIF를 여기에 아래 이름으로 넣으면 코칭 화면에 자동 표시됩니다.
(없으면 우리 스틱피겨 애니메이션으로 폴백.)

## 파일 이름 규칙

```
{운동id}-{성별}.gif
```

- 성별: `male` 또는 `female`
- 현재 운동 id:
  - `neck-side-stretch` (목 옆 스트레칭)
  - `arm-raise` (팔 옆으로 들기)
  - `squat` (스쿼트)

예) `neck-side-stretch-female.gif`, `squat-male.gif`

## 주의
- ⚠️ 라이선스 자산이라 **git에 커밋하지 않습니다**(`.gitignore`로 `*.gif` 제외).
  공개 레포 재배포는 라이선스 위반 위험. 각자 구매분을 로컬/배포 스토리지에 둔다.
- 자세교정 운동(친 턱·견갑 모으기 등)을 추가하면 그 id로 파일명을 맞춘다.
